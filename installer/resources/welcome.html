Google Workspace MCP Server

Complete Google Workspace control through natural language — Gmail, Calendar, Drive, Docs, Sheets, Slides, Forms, Tasks, Chat, Apps Script, and Custom Search.

This installer will:

  • Install the workspace-mcp command to /usr/local/bin
  • Create a Python virtual environment at /usr/local/lib/workspace-mcp
  • Install Claude Code skills (Tufte Classic & CRT design systems)
  • Set up the credentials directory at ~/.google_workspace_mcp
  • Install librsvg for SVG rendering (if Homebrew is available)

Requirements: Python 3.10+ and macOS 13+
